## الخطوة 2: إعداد Google Apps Script

1. من Google Sheet، اذهب إلى **Extensions → Apps Script**
2. سمِّ المشروع: **DonationSystemAPI**
3. احذف المحتوى الافتراضي والصق كود **Code.gs** الموجود في مجلد `apps-script/`
4. غيّر `SHEET_ID` في الكود إلى ID الشيت الخاص بك:

```
كيف تجد Sheet ID؟
افتح الشيت في المتصفح، الرابط يكون:
https://docs.google.com/spreadsheets/d/THIS_IS_THE_SHEET_ID/edit
انسخ THIS_IS_THE_SHEET_ID (النص الطويل بين /d/ و /edit)
```

5. اذهب إلى **Editor** → انقر على **Services** (+) → أضف **Drive API** (v3)
6. احفظ (Ctrl+S)

---

## الخطوة 3: نشر الـ Web App

1. اذهب إلى **Deploy → New deployment**
2. اختر النوع: **Web app**
3. الإعدادات:
   - **Description**: Donation API v1
   - **Execute as**: Me (your-email@gmail.com)
   - **Who has access**: Anyone
4. انقر **Deploy**
5. **授权** (Authorize) - ستظهر شاشة تحذير، اذهب إلى Advanced → Go to project name (unsafe) → Allow
6. **انسخ Web App URL** - ستحتاجه للخطوة التالية

---

## الخطوة 4: ربط التطبيق بـ API

1. افتح ملف **app.js**
2. ابحث عن هذا السطر:
   ```js
   const APPS_SCRIPT_URL = 'YOUR_APPS_SCRIPT_WEB_APP_URL_HERE';
   ```
3. استبدله برابط Web App الذي نسخته:
   ```js
   const APPS_SCRIPT_URL = 'https://script.google.com/macros/s/ABC123/exec';
   ```

## الخطوة 5: النشر على GitHub Pages

1. أنشئ Repository جديد على GitHub (مثلاً: `donation-app`)
2. ارفع الملفات التالية إلى المستودع:
   - `index.html`
   - `style.css`
   - `app.js`
   (لا حاجة لرفع `Code.gs` أو `README.md` أو `DEPLOYMENT.md` إلى Pages - لكن يمكنك رفعها للتوثيق)
3. اذهب إلى **Settings → Pages**
4. اختر Source: **Deploy from a branch**
5. Branch: **main** (أو master) → folder: **/ (root)**
6. انقر **Save**
7. انتظر دقيقة، ثم سترى رابط الموقع: `https://your-username.github.io/donation-app/`

---

## 🎯 اختبار التطبيق

1. افتح رابط GitHub Pages من موبايلك أو متصفحك
2. ستظهر القوائم المنسدلة مليئة بالبيانات من Google Sheet
3. اختر مركزاً → ستظهر المتطوعين تلقائياً
4. املأ باقي الحقول
5. صوّر الإيصال أو ارفع صورة
6. اضغط **تسجيل التبرع**
7. ستظهر رسالة نجاح
8. ارجع إلى Google Sheets → ورقة Submissions → ستجد البيانات محفوظة
9. اذهب إلى Google Drive → مجلد `DonationReceipts` → ستجد الصورة

## ⚠️ ملاحظات مهمة

- **تحديث البيانات**: أي تعديل في شيت Centers أو Volunteers أو DonationTypes سيظهر فوراً في التطبيق (بدون إعادة نشر)
- **النسخ الاحتياطي**: Google Sheets يحفظ نسخ احتياطية تلقائية
- **السرعة**: التطبيق خفيف جداً ويعمل على أضعف الهواتف
- **الخصوصية**: البيانات آمنة في حساب Google الخاص بك، ولا تمر عبر أي سيرفر آخر

## ❓ استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| القوائم فارغة | تأكد من أن Web App منشور ورابطه صحيح في app.js |
| خطأ في رفع الصورة | تأكد من تفعيل Drive API في Services |
| خطأ CORS | استخدم وضع no-cors (مضاف تلقائياً في الكود) |
| الـ Sheet لا يظهر | تحقق من Sheet ID في Code.gs |
