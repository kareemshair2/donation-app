# نظام جمع التبرعات - تطبيق المتطوعين

## 🌐 النظام بشكل مختصر

تطبيق ويب للهواتف (Mobile-first) يهدف لتسجيل تبرعات المتطوعين في مراكز محافظة المنوفية.  
البيانات تُحفظ في **Google Sheets**، ويتم رفع صور الإيصالات إلى **Google Drive**، والواجهة مستضافة على **GitHub Pages**.

```
المستخدم → GitHub Pages (واجهة) → Google Apps Script (API) → Google Sheets + Google Drive
```

## 📱 وصف الواجهة (للهواتف)

- عمود واحد (single-column) ← يناسب كل أحوال الشاشات
- زر submit ثابت في أسفل الشاشة
- حقول إدخال كبيرة ومناسبة للمس
- الكاميرا مفتوحة مباشرة لتصوير الإيصال
- validation فوري قبل الإرسال
- ظهور رسالة نجاح واضحة بعد كل عملية
- القوائم المنسدلة تتحمل تلقائيا من Google Sheets

---

## 👣 خطوات البناء

1. إنشاء Google Sheet بالهيكل المذكور وملئ البيانات التجريبية
2. فتح Tools → Script Editor وإنشاء مشروع Google Apps Script
3. لصق كود Code.gs وحفظه
4. تفعيل خدمة Drive API في Apps Script
5. نشر الـ Script كـ Web App (Execute as: me, Access: Anyone)
6. نسخ رابط الـ Web App
7. تحديث الرابط في ملف app.js (ثابت `APPS_SCRIPT_URL`)
8. رفع ملفات (index.html, style.css, app.js) إلى GitHub Pages
9. اختبار التطبيق مباشرة
